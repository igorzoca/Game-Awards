Mas, o que é o game awards?

O game awards é uma premiação que ocorre no fim de todo ano para coroar os melhores jogos do ano, em variadas categorias.

Nesse site, eu estou abordando apenas a categoria principal, que é a de jogo do ano. Nessa categoria concorrem os jogos Alan Wake 2, Baldur's Gate 3, Marvel's Spider Man 2, Resident Evil 4 remake, Super Mario Bros Wonder e Legend of Zelda: Tears of the Kingdom.
